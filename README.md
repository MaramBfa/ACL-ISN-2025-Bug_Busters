  **Compilation et exécution
1. Se placer dans le dossier racine du projet
   Celui qui contient le fichier pom.xml.
       Exemple (selon votre configuration) :
       cd C:\Users\DELL\Desktop\ACL-ISN-2025-Bug_Busters\ACL-ISN-2025-Bug_Busters
  2. Compiler le projet
   mvn clean compile
3. Lancer le jeu
   mvn exec:java
   
**Environnement de développement
  -Java JDK 17
  -Eclipse IDE 2025-09 (ou version compatible)
  -Apache Maven 3.x

**Fonctionnalités principales
  -Progression sur 10 niveaux fixes avec difficulté croissante
  -Système porte/clé et trésor final au dernier niveau
  -Monstres évolutifs (monstres,fantômes, zombies)
  -Armes différenciées : épée (proximité) et arc (distance)
  -Cœurs de vie ramassables pour regagner de la santé
  -Carte de progression visuelle + HUD amélioré
  
**Notes
  -Le projet suit une méthodologie agile en sprints (4 sprints principaux).
  -Chaque sprint a permis d’ajouter de nouvelles fonctionnalités et d’améliorer le gameplay.
  -Le code est organisé de manière modulaire pour faciliter la modification .
